# ExpressJS Start Project

1. Install express `$ npm install express`
2. Create server file `app.js`
3. How to run `$ node app`, index because the file
4. Install nodemon `$ npm i -D nodemon`, nodemon ini supaya kita tidak perlu nge restart si server filenya, jadi tinggal refresh page aja, pakai nodemon hanya untuk develop not for production
5. run pakai nodemon `$ npm run dev`
6. Coba

- http://localhost:5000/
- http://localhost:5000/about.html
